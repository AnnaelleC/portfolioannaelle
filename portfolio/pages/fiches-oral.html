Oral E5 — Glossaire des termes techniques

🏗️ Architecture & Patterns
MVC
Model–View–Controller. Sépare données, affichage et logique. Évite de tout mélanger.
DAO
Data Access Object. Classe dédiée à la BDD. Changer de BDD sans toucher au reste.
PDO
PHP Data Objects. Connexion sécurisée PHP → BDD. Permet les requêtes préparées.
ORM
Object-Relational Mapping. Objets PHP ↔ tables BDD auto. Doctrine = ORM Symfony.
Requête préparée
Paramètres séparés du SQL → données jamais interprétées comme du code. Anti-injection SQL.
CRUD
Create / Read / Update / Delete. Les 4 opérations de base sur une BDD.
🛠️ Frameworks & Outils
Symfony
Framework PHP professionnel. Structure MVC, routes, auth. Très utilisé en entreprise.
Doctrine
ORM de Symfony. Tables = classes PHP (entités). SQL généré automatiquement.
PHPMailer
Bibliothèque PHP pour envoyer des e-mails depuis un formulaire via SMTP.
DomPDF
Bibliothèque PHP qui convertit du HTML/CSS en fichier PDF dynamique.
PHPUnit
Tests unitaires PHP. Vérifie automatiquement que chaque méthode retourne le bon résultat.
Webmin
Interface web d'admin de serveurs Linux. Modules développés en Perl.
GLPI
Gestion de tickets (incidents, demandes). Simule les demandes clients en entreprise.
AlwaysData
Hébergeur web français PHP + MySQL. Déploiement SSH/FTP. Utilisé pour ARGITIK.
📱 Mobile & Langages
Kotlin
Langage officiel Android depuis 2017. Plus concis et sûr que Java.
SQLite
BDD stockée dans un fichier sur l'appareil. Pas de serveur. Utilisé en Android.
Perl
Langage ancien encore utilisé en admin système Linux. Module Webmin FEBUS.
Linux embarqué
Linux sur appareil spécialisé (boîtier, capteur). FEBUS : boîtiers mesure fibre optique.
XML
Format de fichier structuré. Stocke les données des ateliers dans le projet Intégration.
GitHub Codespace
IDE cloud GitHub. Développer depuis un navigateur. Utilisé en télétravail chez ARGITIK.
📋 Méthodes de projet
SCRUM
Méthode agile en sprints. Planification → dev → revue. S'adapte au changement.
Kanban
Tableau À faire / En cours / Terminé. Outil : ClickUp. Visualise l'avancement.
GANTT
Planning sur une frise chronologique. Répartit les tâches et montre les dépendances.
Product Backlog
Liste des fonctionnalités à dev, classées par priorité. Base du SCRUM.
Test unitaire
Teste une seule méthode isolément. Garantit que les modifs futures ne cassent rien.
SSH / FTP
SSH = connexion sécurisée au serveur. FTP = transfert de fichiers vers le serveur.
🔐 Sécurité web
OWASP Top 10
Classement des 10 vulnérabilités web les plus critiques. Référence mondiale.
Injection SQL
Code SQL injecté dans un formulaire → manipule la BDD. Contre-mesure : PDO.
XSS
Script JS injecté dans une page → vol de cookies. Contre-mesure : htmlspecialchars().
CSRF
Force un user authentifié à faire une action à son insu. Contre-mesure : token CSRF.
RGPD
Loi européenne sur les données perso. Mentions légales, confidentialité, droit à l'oubli.
CVE
Identifiant unique d'une vulnérabilité. Ex : Log4Shell = CVE-2021-44228.
MFA
Auth multi-facteurs (mdp + code SMS). Renforce la sécurité des comptes.
Ransomware
Logiciel qui chiffre les données et demande une rançon. Ex : LockBit.
📦 Vocabulaire Snyk (veille V2)
Bibliothèques (dépendances)
Composants logiciels externes réutilisés dans une app (ex : PHPMailer, DomPDF, Doctrine). Snyk vérifie si elles contiennent des failles connues.
Conteneurs Docker
Environnements légers et isolés contenant une app et tout ce dont elle a besoin. Snyk Container scanne les images Docker pour détecter des vulnérabilités.
IaC
Infrastructure as Code. Fichiers qui décrivent et automatisent le déploiement de serveurs, réseaux, BDD. Snyk IaC vérifie leurs mauvaises configurations.
🛡️ DevSecOps & Réseau
DevSecOps
Sécurité intégrée dans le cycle Dev + Ops. Vérifiée à chaque étape, pas en fin.
Shift Left
Sécurité vérifiée dès l'IDE plutôt qu'en prod. Moins coûteux à corriger tôt.
SCA
Analyse des dépendances open source pour CVE connues. = Snyk Open Source.
SAST
Analyse statique du code sans l'exécuter. Détecte patterns dangereux. = Snyk Code.
SBOM
Inventaire de tous les composants d'une app. Bientôt obligatoire (Cyber Resilience Act).
CI/CD
Pipeline qui teste et déploie automatiquement à chaque commit. Ex : GitHub Actions.
NTP
Network Time Protocol. Synchro horloge réseau, précision ~milliseconde.
PTP
Precision Time Protocol. Synchro haute précision ~nanoseconde. Boîtiers FEBUS.
